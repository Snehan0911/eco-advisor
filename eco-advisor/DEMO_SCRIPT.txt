================================================================================
eco-advisor — 3-Minute Demo Video Script
================================================================================

[0:00 — HOOK]
(pause)
"Hello everyone! Today, I am excited to introduce eco-advisor, a secure, multi-agent AI assistant designed to help individuals and businesses make smarter ecological choices. Finding practical sustainability info—like carbon footprints, localized recycling sorting rules, and eco-friendly swaps—usually takes hours of manual research, but eco-advisor puts all of it in one place."

[0:20 — WHAT IT IS]
(pause)
"eco-advisor is an AI assistant belonging to the 'Agents for Good' track, focusing on environmental sustainability. Built code-first using Google's Agent Development Kit (ADK) 2.0, it connects specialized sub-agents with external computation tools to deliver accurate, non-hallucinated ecological advice."

[0:40 — SHOW THE BANNER]
(show assets/cover_page_banner.png)
"At a glance, the eco-advisor dashboard offers clean interfaces to calculate carbon footprints, look up sorting rules, and recommend sustainable product swaps in a modern, dark-themed dashboard."

[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
"Let's walk through the architecture.
First, every user query starts at the Security Checkpoint node to check for injection attacks and scrub PII like email addresses and phone numbers.
Next, the query routes to the central Orchestrator agent which parses the intent.
If the intent is clear, the Orchestrator routes the query using AgentTools to either the Carbon Advisor or the Recycling Advisor.
If the query is ambiguous, the workflow pauses at the Human node via request_input to ask for user parameters.
Finally, the specialized sub-agents connect to the MCP Server containing the footprint calculator, recycling directories, and product catalogs."

[2:00 — LIVE DEMO]
(switch to playground at localhost:18081)
"Let's run a live demo!
First, we'll ask the agent: 'Calculate carbon footprint for 150 driving miles'.
(pause)
You can see the Orchestrator delegates this query to the Carbon Advisor, which calls the calculate_carbon_footprint MCP tool, yielding an exact 60.60 kg of CO2 without any math errors.
Next, let's ask: 'Suggest an eco-friendly alternative for plastic water bottles'.
(pause)
The Carbon Advisor invokes the search_eco_products tool, suggesting food-grade stainless steel or glass bottle alternatives.
Finally, we query: 'Can I compost a paper towel?'.
(pause)
The Recycling Advisor checks the composting guidelines and immediately confirms it's brown compostable and gives shredding advice."

[3:00 — SECURITY & MCP]
(pause)
"The security checkpoint ensures all prompts are safely sanitized and logged to a structured JSON audit trail, while the MCP server guarantees that math and data lookups are offloaded to hardcoded algorithms rather than unpredictable LLM generations."

[3:30 — IMPACT / CLOSE]
(pause)
"eco-advisor empowers users to make positive environmental choices every day. By providing instant, clear, and verified answers, it helps lower the barrier to sustainable living and zero-waste habits. Thank you!"
